 # 2019，我想和爸妈谈谈


### 又要过年了，每一次过年，我都不是那么期待。因为好像家里即便是过年，也没有欢快的气氛，也很难一家人一起做饭，然后一起吃饭。虽然这一次，我仍然不敢抱有多大的期待，但是我觉得我现在已经有足够的心理能量了，我想有些事情，要敞开来谈一谈。

 

### 其实过去几年，有很多次，我都想这样谈一谈，但经常写着写着自己就控制不住，自己就泪流满面。而且特别害怕自己写的内容最后变成一种控诉，一种抱怨，一种指责。这一次，我想自己比之前更加成熟了，也更加敢于承担了。但是毕竟，我是家庭的一份子，里面包含了太多的感情因素，我也无法做到100%的客观和公正。
       
       
### 所以，为了保持更加的冷静和客观，我采取第三人称的方式。当然，如果里面还能读出些许的抱怨成分，可能还包含着我的不成熟。希望爸妈不要把它放大。我的初心只有一点，希望我们这个家能变得好一些。

 

### 想一想过去的20多年，很多事情已经淡忘，但也有一些事情还是会深深地刻在脑海里。还是再强调，之所以回顾这些事情，不是为了指责，而是说一些我的感受，以及希望父母能够多少有些反思和改变，哪怕只是一点点。

 

## 关于爸爸，清楚地记得几件事。

1.不允许我带小朋友到家里玩，嫌吵闹

2.年轻时爱睡懒觉

3.小时候被爸爸大声训斥，不让哭出声音

4.骑摩托带我出去玩

5.不辞辛苦，骑摩托送我上学。带我吃好吃的

6.在奶奶眼里，爸爸什么都是好的。

7.爸爸受伤住院，我其实特别想关心，问候，但是由于从小和爸爸之间但交流模式，我还是没有勇气去突破

8.记得虽然自己考了不好的大学，爸爸，一个不爱干活的人，大汗淋漓地帮我拎着行李箱爬楼

9.为了挣钱，不顾身体，研究彩票。自己辛苦的挣钱，给我和妹妹。

10.但是好像有点活得只是为了挣钱，每天只是坐在电脑前。甚至不顾自己的身体健康。

11.思维活跃，思想开放，在遇到大问题的时候，还是要靠爸爸。

 

## 关于妈妈

1.爱叫，音调高而急

2.小时候，妈妈爱掐我们，捏一下很疼

3.勤劳，一个女人，自己除茅厕，背着那么重的药桶打药。大夏天去拾棉花，拾麦子。

4.去姥姥家都是自己带我们，几个人骑着车子，特别费劲地长途跋涉，尤其记得有一年大雪天。。

5.爱吐槽，自己无力改变，吐槽完了继续不改。

6.情商低，不懂妥协，又逆来顺受

7.不爱动脑子，不爱思考

8.比较勤奋，爱早起，爱锻炼


9.做饭不好吃，做事情毛躁。和性格有关，和家庭环境也多少有关。


## 关于他俩（有一些事不能放在一个人身上，就合并说）


 1.大概是7岁的一个大雪天，爸爸妈妈打架，我吓的边哭边往奶奶家跑，路上还摔倒了，我从来没和他们说过
 

2.我很少和家里打电话，一方面为了省钱，一方面好像真的没话说，因为只要想起家，就是很多不好的回忆；

3.现在有了微信以后有偶尔的联系，也因为我的成长，可以很客观地看待一些事情，并尽可能地回忆父母的好，以及感恩他们对我的养育之恩。


## 总之，

### 从小在心里对爸爸形成的印象：严肃，严厉，爱板着一张脸，懒惰，思想开放

### 从小在心里对妈妈形成的印象：勤劳，质朴，倔强，不懂爱惜自己，不会说话，情商极低

 

### 首先，非常感恩爸爸妈妈给了我受更好教育的机会，对我的信任。我知道他们是给了我他们所能给的最好的教育，最好的关心。
他们也是平凡人，作为一个个体，也有自己认知的限制，有自己的情感，有自己的故事。在他们所受的教育中，也是不断成长和不断变化的。所以，我一定要说清楚，以下我写这些，只是想通过阐述，希望他们有所反思和有所改变，而不是让他们自责。也不是怨恨他们。只是希望有所改变后，对于弟弟妹妹能有更好的影响，能让他们感受到来自家庭的爱，从而让他们的人生变得更好。如果遇到挫折，也不要是因为家庭的原因。

## 这一切的开始，就是爸爸妈妈之间的感情。

### 小时候，妈妈向我吐槽，和爸爸之间的吵架，但是又逆来顺受，每一次又在妥协，又在继续忍受。不懂得爱惜自己，说白了，就是卖力还不讨好。。

### 爸爸，作为一个男人，从不诉说自己心里是如何想的，按照传统的男人的角色，他也不可能向我们诉说。但是，作为一个丈夫，我无法给他及格分；作为爸爸，可能勉强及格，作为儿子，可能勉强及格。每一次交流，也只是独白式的和我们交流，从来不听我们的想法，只是觉得我们幼稚。

### 印象深刻的一次，我想买书的封皮，还有本子的夹子。爸爸说没用。

### 经验丰富，见的多，但是陷入了盲目自信，经常觉得自己的都是对的，容不得别人半点的疑问。

### 懒惰，不承担家庭的责任。但是一方面，也让我感受着对我的信任，对我的培养，对我的疼爱。

## 所以，小时候，我同情妈妈，怨恨爸爸。

 

### 从客观来讲，妈妈不应该向一个小孩子吐槽，但是作为一个普通的家庭妇女，她的认知水平有限，交朋友的范围也有限，带着传统的观点，家丑不可外扬，好像他也只能和我说那些她受的委屈。所以，我现在写出来，不是怪她，也希望他不会怪自己。也希望爸爸也能体谅她。但由于这种单方面的吐槽，和当时我作为一个孩子认知发展的有限，让我从心里对爸爸形成了怨恨和不理解。同时，由于爸爸的严厉，和爸爸之间的距离本身就远，没有和爸爸沟通和交流的机会。于是恶性循环，变得更远，更有隔阂，更没有交流的空间。

 

## 第一阶段：大学之前，不理解，怨恨。

### 从某种程度上，这种影响，也或多或少影响了我的学习。因为在内心深处，有太多的不理解，太多的苦。因此会有分心。不能完全无忧无虑地投入学习。每次放假回家，感受到的是一个沉默的家庭，一个只要一说话就开杠的家庭，一个充满了火药味的家庭。也是更加的沉重。记得还存在过一段和爸爸的“冷战”。

 
## 第二阶段：大学之后，客观看待，但无力改变

### 后来长大了，学了一些知识。这些知识有好的一方面，也有不好的一方面。就是因为学了这些知识，让我更加了解一个家庭对一个人成长的影响，，所以曾经，我更加的无奈，尤其是每次看到同学和爸妈之间朋友一样的交流，看到别人一家子其乐融额的在一起的时候，我是更加的无奈，更加的悲凉。但是工作之后，我读了很多书，一下子又提高了一个高度。家庭对自己的影响再大，也大不过自己。一个人的一生会有很多经历，童年经历，受教育能力，工作经历等等。但是我们是面向未来的，我们自己要对自己的人生负责。所以，我更加看开了，也在不断学习，让家庭对自己的负面影响降到最低，并更多的用自己所学，用自己的能力来感恩父母，孝敬父母。我还写了一篇文章‘阿德勒心理学解决了困惑我多年的问题“https://www.jianshu.com/p/cccdfee9d641。收获了很多好评。

 ### 我不再单方面的怨恨爸爸，我想，他也有他的苦衷。但由于心里能量不足，我依然没有办法和爸爸来敞开谈论这个话题，我怕话题刚一开始，就是满满的委屈，就会泪流不止，因为眼泪里包含的是从小到大，十几年的不理解，十几年的距离感，虽然没有了恨，还有十几年的怨。

### 于是我单方面想启发妈妈，让她能有所改变。可是一个人的认知几乎是很难改变的，事实也证明如此。虽然现在矛盾少了，但她所采取的是逃避。爸爸只是觉得妈妈是喜欢玩才每天出去，不知道他是真不清楚还是假的，其实妈妈是太压抑了，包括妈妈嘴上的一块黑色的，心理影响生理，我觉得也是由于心情长期压抑所致。



## 第三阶段：依然能量不够，但却希望尽自己一份力

### 虽然我长大了，我学了很多相关但知识，能更加全面地，深刻地看待家庭存在的问题，并尽可能的让他对自己的不良影响降到最低。尽可能的来记住父母对我的养育之恩，以及不断丰富自己，今早对他们尽孝。把他们当孩子看待，希望他们能收获内心的平和。

### 但是，我还有弟弟和妹妹。他们现在，依然还在我的第一阶段。而我深知第一阶段是多么困难，不仅是自己在挣扎，一旦无法转变，将永远怀着对家庭的恨，最可怕的是，会把这种恨转移到自己的工作和生活，甚至自己未来的家庭。


 

### 所以，我想尽自己的一份力，这一次，不是给妈妈，而是给爸爸说一下我的心路历程和我的想法。

### 因为我发现，爸爸是源头，爸爸如果能转变，才能带动整个家庭的转变。爸爸如果能放下自己内心的那些积怨（虽然我并没有听他讲过自己到底如何想的，但是我觉得他心里肯定也是有怨的，有自己的苦），如果能放下，如果能超越于个人，如果能多关心一些妈妈，能多主动和弟弟妹妹交流一下，可能会有意想不到的效果。

### 但是，我并不敢奢望，期望越大失望越大。我只是尽力告知我最真实的想法，决定还是在爸爸自己。

### 而且，说一些很多人忌讳讨论的话题，爸爸妈妈年纪越来越大。他们结婚也有30年，过去的30年，存在太大的缺憾，看似好像没有什么必要要改善，但是哪怕只是还有十年，也并不是一个短暂的时间，我还是希望最后的最后，每个人心里都能充满爱。

### 对于我们孩子来讲，也希望父母不带缺憾的离开，也希望自己能在父母的老年，能特别和谐的尽自己的孝心。
### 而且，最近妹妹出了事情，我还是发现，爸爸妈妈在用固有的思维方式处理问题，并且只是觉得是孩子的问题。同样作为孩子，我其实不适合说这些话，因为说出来好像是在指责父母。但我也不得不客观的说一些事实，妹妹从小的成长就是东躲西藏，尤其是爸爸，我想让他回想一下，自己对妹妹倾注了多少关心。就在爸爸说妹妹好像没见过男人这句话的时候，我觉得是巨大的讽刺，是的，恰恰是他从小缺少了父爱，导致了现在的性格。我不能说这是唯一因素，但是真的不可以忽略，我希望爸爸能好好回想一下。

### 其实很多家庭都有自己的问题。我自己虽然是学心理学的，但主攻方向并不是心理咨询。一方面，由于我自己的心理能量不足，不想去学习相关的知识。更重要的是，心理咨询有“熟人不咨”的原则，心理咨询师如果给自己的亲朋好友做咨询，会产生双重关系或者多重关系。就像我文中一直强调的，我不可能做到完全的中立和客观。我连自己的家庭都无法帮助，我哪里来的能量去给被人做咨询呢？不过学习心理学也带给了我帮助，让我进行自我修复。所以，我现在也才能尽自己的力，和爸妈谈一谈，希望家庭能有所转变。哪怕只是还有十年，我也不希望最终是草草收场，爸妈就这么带着遗憾过完一辈子。
   

   作为一个还在为钱奋斗的家庭，我们也不可能去花钱去咨询，来修复自己的家庭关系。那就任其发展吗，就像我自己可以修复一样，我想我们可以自己帮助自己。虽然爸爸妈妈没有学习过心理学，但是最基本，最朴素的道理和情感是有的，人心都是肉长的。生命是短暂的，也是漫长的。我特别的希望，在人生的最后一个三分之一，他们能对彼此好一些。而不是像现在一样，好像一对“假夫妻”。哪怕是真的没有感情，至少在生活上，对彼此多一些关心，多一些关注。而不是像机器人一样，只知道挣钱。

更重要的是，我希望弟弟和妹妹可以收到来自家庭真正的爱，在以后自己建立家庭的时候，能摆脱之前不好的家庭模式的影响。


其实，我想写的完整一些，客观一些，理智一些，但确实还是会有主观的成分，因为毕竟我也是家庭的一份子，哪怕我的修为再高，也无法摆脱。更何况，我觉得我还是没有完全地理智。而且写得好像也不是特别有条理。因为是一气呵成，我不想再过多的去回顾，去撕开伤口。只要能表达清楚我的意思，我想就可以了。这一次不想要求完美了，因为每一次的要求完美，这封信就会迟到一年。为了尽早地表达我的想法，就这样不完美地发出来吧。
